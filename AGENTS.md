# Showtime Website Project Memory

## Project Overview

This repo is a small static Astro site for ShowTime Lite, a browser-based presentation tool. It uses Astro 4 with Tailwind CSS and builds to static HTML in `dist/`.

## Commands

- `npm run dev` starts the local Astro dev server.
- `npm run build` generates the static site.
- `npm run preview` previews the generated build.
- `npm run lint` is defined as `astro lint`, but verify availability before relying on it.

## Current App Structure

- `src/layouts/Layout.astro` provides the HTML shell, metadata, favicon, dark body styling, and footer.
- `src/pages/index.astro` is the main landing page. It imports `Hero` and `DemoPreview` and currently contains the hero, demo preview, roadmap/product status, and merged pricing/access sections inline on the home page.
- `src/pages/demo.astro` is a separate interactive demo page. It embeds `https://showtime-lite.vercel.app/demo` in an iframe and has simple top navigation.
- `src/components/Hero.astro` contains the fixed top header/nav and hero copy.
- `src/components/DemoPreview.astro` displays `/st-lite_screenshot.jpg` and overlays a link to `/demo`.
- `src/components/Logo.astro` is an inline SVG logo.

## Styling And Assets

- Tailwind config lives in `tailwind.config.cjs`.
- Custom colors are `primary: #00a8cc`, `secondary: #e0e0e0`, and `dark: #0c0c0c`.
- The site uses a dark, minimal visual style with cyan primary accents.
- Public assets include `logo.svg`, `icons.svg`, `favicon.svg`, `favicon.png`, `appicon.png`, and `st-lite_screenshot.jpg`.
- `public/demo-preview.gif` currently appears to be placeholder text, not a real GIF.
